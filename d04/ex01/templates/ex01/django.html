Ex01 : Django, framework web.

Django est un framework web en Python créé en 2005, qui permet de développer rapidement des sites web sécurisés et maintenables.

Il utilise le pattern MTV (Model-Template-View) et facilite la gestion des bases de données, la sécurité et les templates HTML.
